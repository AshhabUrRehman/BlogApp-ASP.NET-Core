# Blog-App System

A simple blog web application built with ASP.NET Core MVC with role-based authorization. Users can read blog posts and comment, while only admins can create, edit, or delete posts. The app includes user authentication and a responsive UI.

## Features
- Role-based access:
  - Users: Read posts and add comments
  - Admin: Create, edit, delete posts, Image uploads for posts
- View posts by category
- User registration and login
- Responsive design using Bootstrap
- Password hashing, role-based authorization, secure login/logout

## Technologies Used
- ASP.NET Core MVC
- SQL Server (local)
- Entity Framework Core
- HTML, CSS, Bootstrap
- ASP.NET Identity, role-based authorization
